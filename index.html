Projects

Describe the kinds of projects you like to work on and give a brief overview of how you accomplished them...

Project #1 Strava Commute Marker

The main idea of the project is to simplify the process of marking the commutes for Strava athlets. Now it has to be done manually after the activity is finished. Commuting to the school or office is something people do everyday, so the process of marking it becomes bundersome. As a result, city authorities get incomplete data about the popularity of commute routes and lose the opportunity to react properly.

The web application is still under development and I am planning to add more features and improve the UI/UX in the nearest future. The project is built using Ruby on Rails. Its pilot version was deployed to AWS.
